# Advancements Search [NeoForge Port]

A NeoForge 1.21.1 port of the original Fabric mod
[Advancements Search by diskria](https://github.com/diskree/AdvancementsSearch).

Adds a search bar to the in-game advancements screen so you can quickly find
any advancement by title, description, or icon name. Compatible with
[Better Advancements](https://www.curseforge.com/minecraft/mc-mods/better-advancements)
when present.

> ⚠️ **README WIP** — full feature documentation, screenshots, installation
> instructions, and Japanese version coming soon.

## Building from source

1. Install JDK 21
2. Download `BetterAdvancements-NeoForge-1.21.1-0.4.3.21.jar` from CurseForge
   and place it in `libs/` (see `libs/README.txt`)
3. Run `./gradlew build`
4. Find the built jar at `build/libs/advancementssearch_neoforge-*.jar`
